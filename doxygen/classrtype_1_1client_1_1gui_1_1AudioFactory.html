◆ getMusicPath()

std::string rtype::client::gui::AudioFactory::getMusicPath ( MusicId  id)
static

Return the filesystem path for a music identifier.

The returned path is suitable to pass to MusicManager::loadFromFile (or directly to SFML) and is relative to the project root. The factory currently returns .mp3 files; change the implementation if you want to move to .ogg or configurable assets.

Return the filesystem path for a short sound effect.

The returned path can be passed to sf::SoundBuffer::loadFromFile. Prefer using the SoundManager helper which loads and stores buffers centrally and prevents lifetime issues.
