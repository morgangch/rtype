Detailed Description

Component storing room metadata on the server.

This component is attached to a room-entity and contains properties used by the server to manage matchmaking, visibility and game lifecycle for the room.

Example uses:

  • When broadcasting room lists to clients
  • When deciding which players receive in-game state updates
  • Determining who is allowed to perform admin actions in the room
