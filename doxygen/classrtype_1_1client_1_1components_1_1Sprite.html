Detailed Description

Component for visual representation of entities.

The Sprite component contains all data needed to render an entity visually using SFML. It supports both textures and basic colored shapes.

Example usage:

// Create a sprite with texture
world.AddComponent<Sprite>(playerEntity,
"assets/sprites/player.png",
sf::Vector2f(32.0f, 32.0f));
// Create a colored shape (fallback)
world.AddComponent<Sprite>(enemyEntity,
sf::Vector2f(24.0f, 24.0f),
sf::Color::Red);
