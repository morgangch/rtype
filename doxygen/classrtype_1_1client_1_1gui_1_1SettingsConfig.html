Detailed Description

Manages user settings persistence in JSON format.

Features:

  • Load settings from config.json
  • Save settings to config.json
  • Default values if config file doesn't exist
  • Keybinds mapping (Up, Down, Left, Right, Shoot)
  • Network settings (IP and Port)
  • Accessibility settings (Daltonism mode)

JSON Structure: { "keybinds": { "up": 22, // W key "down": 18, // S key "left": 0, // A key "right": 3, // D key "shoot": 57 // Space key }, "network": { "ip": "127.0.0.1", "port": "4242" } }
