Detailed Description

Music management system for the game.

This file implements the MusicManager class which provides a simple wrapper around SFML's sf::Music to handle loading, playback, volume, and mute functionality. It maintains internal state to allow easy control of background music in the game.

Features:

  • Load music from file
  • Play/stop music with optional looping
  • Set and get volume (0 to 100)
  • Mute and unmute music, remembering volume level
  • Check if music is loaded and currently playing
