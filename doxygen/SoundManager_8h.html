Detailed Description

Small helper that owns and plays short sound effects (SFX).

SoundManager centralizes loading, storing and playing of small sound effects. It keeps ownership of sf::SoundBuffer instances so that sf::Sound objects can safely reference them. Use this class instead of having many independent sf::SoundBuffer/sf::Sound pairs across the codebase.
