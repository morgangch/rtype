Detailed Description

Core packet structures for network communication.

Author
R-Type Team
Date
2025

This file defines the fundamental packet structures used throughout the R-Type networking system. It provides the basic building blocks for reliable network communication including headers and data payloads.

typedef struct packet_header_s packet_header_t

Network packet header structure.

Contains all metadata required for reliable packet transmission including sequence numbers, acknowledgments, authentication, and routing information. The header is transmitted with every packet to enable proper networking features like reliability, ordering, and security.

typedef struct packet_s packet_t

Complete network packet structure.

Represents a complete network packet consisting of a header with metadata and a data payload. This is the primary structure used throughout the networking system for all packet operations.
