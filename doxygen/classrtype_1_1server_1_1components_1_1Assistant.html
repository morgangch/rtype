float rtype::server::components::Assistant::shootCooldown = 0.0f

Seconds remaining until next allowed shot.

Initialized to 0.0f (can fire immediately). The server system decreases this value by delta time and sets it to a positive value after firing (e.g. 0.6f) to implement a firing rate.
