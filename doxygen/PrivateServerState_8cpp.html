Detailed Description

Implementation of the private server connection state for the R-Type client GUI.

This file contains the implementation of the PrivateServerState class, which handles the private server connection interface. Users can either join an existing private server using a 4-digit code or create a new private server as an admin.

Key features:

  • Server code input validation (4-digit numbers 1000-9999)
  • Join existing private server functionality
  • Create new private server with random code generation
  • Text input handling with cursor animation
  • Real-time input validation and feedback

The state transitions to PrivateServerLobbyState upon successful server connection or creation.
