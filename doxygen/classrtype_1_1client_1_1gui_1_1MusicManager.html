Detailed Description

Simple music manager wrapper around sf::Music.

Responsibilities:

  • Load music from a file path (streamed from disk).
  • Play / stop and looping control.
  • Volume control in the range [0..100].
  • Mute semantics that remember the previous volume and pause/resume playback when muted/unmuted.

The class intentionally keeps a minimal state model (loaded, volume, muted) so UI components can query the current state and present accurate controls to the user (e.g. mute button icon, volume slider).

Mute or unmute playback.

When muted the manager will pause playback (if playing) and preserve the configured volume value. Unmuting restores the previous volume and resumes playback only if it was playing before muting.

Parameters
