◆ g_stateManager

StateManager * rtype::client::gui::g_stateManager = nullptr

Global accessor for the active StateManager instance.

Some subsystems (for example network controllers) run as free functions and need a way to request a state change. The application sets this global pointer once during initialization (see main.cpp).
