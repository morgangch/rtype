Detailed Description

Singleton that manages colorblindness simulation state and shader.

The Accessibility manager stores the currently selected daltonism mode and exposes a shader configured with the appropriate uniforms to recolor the whole frame. Call getShader() each frame you want to apply the effect; it will return nullptr if no mode is active or if the shader failed to load.
