Detailed Description

Singleton class for centralized font management.

The FontManager class provides a centralized way to load and access fonts throughout the GUI system. It implements the singleton pattern to ensure only one instance exists and fonts are loaded only once.

Features:

  • Platform-specific font fallback system (Linux, macOS, Windows)
  • Automatic font loading with error handling
  • Single point of access for all GUI components
  • Memory efficient (fonts loaded once and reused)

Usage:

const sf::Font& font = FontManager::getInstance().getDefaultFont();
text.setFont(font);
